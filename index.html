Lazydoro

This is my personal entry to the Hackaday.io Trinket Everyday Carry Contest. The goal is to build a simple activity tracker, using an Adafruit Pro Trinket, and an accelerometer, that will notify me if I'm sitting for too long and nudge me to take a 1min walk every 30 minutes.

Graphs

In the current form, this project onsy holds the data I've gathered so far (sitting for a few minutes and working on my computer), in Graph form. You can see that here.
